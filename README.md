# Stock-Portfolio-Manager-C++

A simple console-based Stock Portfolio Management System built using C++.
This project allows users to buy stocks, sell stocks, track holdings, and calculate total portfolio value.

🚀 Features

✅ Buy stocks

✅ Sell stocks

✅ Automatically update stock quantity

✅ View complete portfolio

✅ Calculate total portfolio value

✅ Menu-driven console interface

🛠️ Tech Stack

Language: C++

Concepts Used:

Object-Oriented Programming (OOP)

Classes & Objects

Encapsulation

Vectors (STL)

Menu-driven system

Basic financial computation logic

📂 Project Structure

The system contains two main classes:

1️⃣ stocks Class

Handles:

Stock name

Quantity

Price

Portfolio value calculation

2️⃣ classification Class

Manages:

Buying stocks

Selling stocks

Storing stock records

Displaying portfolio

Calculating total portfolio value

📊 How It Works

User selects an option from the menu:

Buy stock

Sell stock

Show portfolio

Show total value

When buying:

If stock already exists → quantity increases

If new stock → added to portfolio

When selling:

Quantity is reduced (if sufficient shares available)

Total value is calculated as:

Total Value = Quantity × Price (for all stocks)
